# MVC4Fun
功能:支援中文和英文的 電影推薦清單，可新增、刪除、修改、查詢

## <建立資料庫>
### 需至Package Manager Console 視窗 key  <br/>
  1.add-migration initial 
   <br/>	<sub>請確認是否有成功，在進行第2 段 </sub><br/>
  2.add-migration SeedData
    <br/><sub>請確認是否有成功，在進行第3 段</sub> <br/>
  3.update-database
    <br/> <sub>完成後會出現成功!</sub>
